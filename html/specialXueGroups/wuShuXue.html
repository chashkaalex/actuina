Архаические точки

Архаические точки (или точки у-син, шу-точки, у-шу-сюэ). Точки открытые и использовавшиеся ранее прочих. Имеют “ирригационные” наименования и принадлежность у-син. Эту группу делят на две – для цзан и фу органов. На каждом канале-цзин расположены пять у-шу-сюэ:

  1. Исток (цзин - “колодец”). Все истоки - для неотложной помощь. Применяется в критических состояниях для ци или шень. Для органов цзан, их син - дерево, а для органов фу - металл. Все точки-истоки, кроме истоков каналов синь-бао-ло и почек, находятся возле угла ногтя. Исток синь-бао-ло - на кончике среднего пальца, исток канала почек - центр стопы (юн-цюань).
  2. Ручей (ин). Все ручьи - для болезней со свойствами жара. Для органов цзан, их син - огонь, а для органов фу - вода.
  3. Быстрина (шу). Все быстрины - для болей в суставах и тяжести в теле. Для органов цзан их син - земля, а для органов фу - дерево.
  4. Река (цзин). Все реки - для наружных синдромов, в числе симптомов которых лихорадка, озноб, кашель или одышка. Для органов цзан, их син - металл, а для органов фу - огонь.
  5. Устье (хэ). Все устья - для болезней всех органов фу вообще и, в частности, для болезней желудка (то-есть для всех болезней связанных с неправильным питанием). Также, устья используются при извращениях [хода] ци. Для органов цзан, их син - вода, а для органов фу - земля.

Говорят, что энергия вытекает из истока, просачивается в ручей, затем стекает по ручью в быстрину, течёт по реке и впадает в устье. То-есть, энергия течёт от малого к большому, от мелкого к глубокому, от дальнего к близкому. У-шу-сюэ являются отправной точкой для расчета точек связанных с астрологией. При работе с у-шу-сюэ пользуются принципами у-син.
